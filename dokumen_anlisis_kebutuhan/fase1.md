 Problem Statement

Sistem absensi berbasis aplikasi di MTsN 1 Yogyakarta belum berjalan optimal karena sering mengalami error saat digunakan. Kondisi ini menyebabkan proses pencatatan kehadiran siswa tidak dapat dilakukan secara konsisten melalui sistem, sehingga guru kembali menggunakan metode manual berupa pencatatan di spreadsheet dan pengiriman laporan melalui grup komunikasi.
Proses manual tersebut menimbulkan beberapa permasalahan, seperti keterlambatan rekap data, potensi kesalahan pencatatan, duplikasi data, serta ketidakefisienan dalam pengolahan informasi kehadiran siswa, terutama saat diperlukan rekapitulasi untuk laporan akhir seperti pembagian rapor.

Kebutuhan fungsional 

Sistem harus dapat digunakan guru mata pelajaran untuk mencatat kehadiran siswa secara langsung setiap pertemuan.
Sistem harus dapat digunakan guru mata pelajaran untuk mengirim data absensi tanpa melalui media komunikasi terpisah.
Sistem harus dapat digunakan guru piket untuk mengakses seluruh data absensi dari semua kelas secara terpusat.
Sistem harus dapat digunakan guru piket untuk merekap data kehadiran siswa secara otomatis.
Sistem harus dapat digunakan wali kelas untuk melihat rekap kehadiran siswa dalam periode tertentu.
Sistem harus dapat digunakan wali kelas untuk memperoleh laporan kehadiran siswa sebagai bahan pembagian rapor.
